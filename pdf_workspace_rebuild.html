      ツクエノウエの道具箱 操作説明書

      1. アプリ概要

      ツクエノウエの道具箱 は、ブラウザ上でPDF作業を行う統合ツールです。

      主なタブは以下です。

      • PDFワークスペース
      • OCR抽出
      • 検索可能化
      • QR作成

      2. 事前確認

      1. 推奨ブラウザは最新版のChrome/Edge。
      2. 大きいPDF処理時はPCメモリに余裕がある環境を推奨。
      3. 初回読み込み時やOCR系では通信環境が必要になる場合があります。
      4. 保存ファイルはブラウザのIndexedDBに保持されます。

      3. PDFワークスペース

      3-1. 基本フロー

      1. PDF / JPG / PNG を追加へドロップまたはクリックでファイル追加。
      2. ページを選択・並び替え・回転・削除。
      3. 右カラムで出力設定を調整。
      4. 全体を保存または選択を保存で出力。

      3-2. ページ選択

      1. 単一選択: サムネイルをクリック。
      2. 範囲選択: Shift + 左クリック。
      3. 範囲解除: 選択済み範囲に対して Shift + 左クリック で同様に解除可能。
      4. 複数選択後はまとめてドラッグ移動できます。

      3-3. サムネイル上の操作

      • ↻ で回転。
      • ✕ で削除。
      • … で補助メニュー（先頭へ移動/末尾へ移動など）。
      • サムネイル画像をダブルクリックで拡大プレビュー。
      • プレビューは×、背景クリック、Escで閉じます。

      3-4. ツールバー

      • かんたん操作: 全選択、解除、選択回転、選択削除、保存系。
      • 詳細操作: 見開き分割、見開き結合、画像保存、Undo/Redoなど。
      • 直前操作はスナックバーの元に戻すでも復帰できます。

      3-5. 右カラム設定

      出力設定

      • 余白調整
      • ページ番号（開始/終了、位置、フォント、サイズ、X/Y補正）

      見開き設定

      • 綴じ方向
      • 結合1ページ目の扱い

      保存ファイル

      • 展開: IndexedDBから再展開
      • 削除: IndexedDBから削除

      3-6. 内部機能

      スタンプ
      テキストスタンプ、標準スタンプ、白マスキングを配置して保存。

      圧縮／漂白
      プレビュー確認後に選択ページへ適用し、差し替え保存。

      診断
      保存状態確認、DB操作の補助。

      4. OCR抽出タブ

      1. OCR対象PDFを追加。
      2. 言語・解析条件を設定。
      3. 実行後、抽出結果を確認してTXT保存。
      4. 大量ページは時間がかかるため、ページ範囲を絞る運用を推奨。

      5. 検索可能化タブ

      1. PDFを追加。
      2. 言語・解析倍率・ページ範囲を設定。
      3. OCRで透明テキストを付与した検索可能PDFを生成して保存。
      4. OCR抽出より重い処理になるため、先に少ページで検証を推奨。

      6. QR作成タブ

      1. テキスト/URLを入力。
      2. サイズ等の設定を調整。
      3. QRを生成し、画像として保存。

      7. トラブル時の確認

      OCR実行時エラー
      通信状態、ブラウザ拡張（スクリプト遮断）、再読込を確認。

      保存が開始しない
      ブラウザのダウンロード制限/ポップアップ制限を確認。

      動作が重い
      ページ数を分割して処理、不要タブを閉じる、再読込後に再試行。

      状態が不整合
      保存ファイルから再展開、必要に応じて診断機能を使用。

      8. 推奨運用

      1. 先にPDFワークスペースで並び替えと回転を完了。
      2. 次に必要ならスタンプまたは圧縮／漂白。
      3. 最後に保存形式（PDF/画像）を確定。
      4. OCR系は最終成果物に対して実行すると手戻りが少ない。
